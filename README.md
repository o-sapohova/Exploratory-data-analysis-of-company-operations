# Exploratory Data Analysis of Company Operations

## Project Overview
This project presents an **Exploratory Data Analysis (EDA)** of company operations.  
The analysis covers:
- Sales: revenues, expenses, profits, and product popularity.
- Order processing time.
- Sales dynamics over time.

The goal is to identify key patterns, challenges, and opportunities for improving business processes.

---

## Tools and Libraries
- **NumPy** – numerical computations
- **Pandas** – data manipulation and cleaning
- **Matplotlib** – data visualization
- **Seaborn** – advanced statistical visualizations

---

## Workflow
1. **Data overview** – initial inspection of structure and content.
2. **Data cleaning** – handling missing values, duplicates, and formatting issues.
3. **Sales analysis** – revenues, expenses, profits, product popularity.
4. **Process analysis** – order processing times.
5. **Sales dynamics** – trends and seasonality.
6. **Visualizations** – charts and plots for better insights.
7. **Conclusions & recommendations** – actionable suggestions for business improvement.
